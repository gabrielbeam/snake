#How to run

### Docker

Build the Docker image

```shell
`docker build -t snake .`
```

Run the image
```shell
`docker run -p 8080:8080 -t snake`
```

This will launch the application on port 8080

`http://localhost:8080`

### Intellij IDEA

Click on the play button next to `main` method on `Application.kt`
